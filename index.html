About

My name is Marek Fišera. I'm from Czech Republic, last few years I spent in Prague. In summer of 2011 I completed my Bachelor degree at CTU, Faculty of Electrotechnics, Software Engineering. In next school year I started my Master degree at the same university.

Since 2007 I'm working as a developer of bussines solutions/web sites for travel agencies, first two years as XHTML/CSS/Javascript coder and since 2009 as .NET developer.

Neptuo OS

This is my Bachelor thesis written in summer semester of 2011 at CTU. It is written in Java EE and consists of two web modules, one for client app, using GWT. And one for server app. Server side app is service-oriented, service layer is realised by neptuo-service framework.

For more information see neptuo.com/cs/projects/os or check beta version at beta.os.neptuo.com/webdesk

is4wfw

This is information system written in PHP. It supports many features as I continously develop it in my spare time for three years. It has advanced sport-library for storing information about sport matches, article management, tree-like structure for page rendering, etc. Some projects are using it, more information can by found at is4wfw.neptuo.com.

This web is also built using is4wfw.

Windows Dock

This is a MAX OSX like application launcher for Windows. It is written in Microsoft's WPF (Windows Presentation Foundation). This was my first project in WPF. It is also very customizable and still under development of new features. I'm using it on Windows 7.

For more information, check Windows Dock@Google Code, where you can find actual stable version and notes about each release.
